# facial_recognition_homework

这段代码的核心思路是利用深度学习模型（特别是预训练的人脸识别模型）来比较两张人脸图像是否属于同一个人。它通过以下步骤实现这一目标：

1.加载预训练模型
• 使用TensorFlow加载一个预训练的人脸识别模型（通常是基于卷积神经网络的模型，例如FaceNet）。
• 模型文件是一个`.pb`格式的二进制文件，包含模型的结构和权重。
• 加载模型后，获取模型的输入张量、输出张量和训练阶段标志张量，以便后续使用。

2.图像预处理
• 对输入的人脸图像进行预处理，使其符合模型的输入要求。
• 调整图像大小为模型指定的输入尺寸（例如160x160像素）。
• 对图像进行归一化处理，将像素值从[0,255]缩放到[-1,1]范围。
• 在图像数据的第0维添加一个额外的维度，使其形状从`(160, 160, 3)`变为`(1, 160, 160, 3)`，以满足模型输入的要求。

3.计算人脸嵌入向量
• 使用预训练模型将预处理后的图像转换为一个固定长度的嵌入向量（embedding）。
• 嵌入向量是模型的输出，它将人脸图像映射到一个高维空间，使得同一人的嵌入向量在空间中更接近，而不同人的嵌入向量更远离。

4.比较人脸嵌入向量
• 计算两张人脸图像的嵌入向量之间的相似度或距离。
• 使用欧氏距离（Euclidean Distance）来衡量两个嵌入向量之间的差异。
• 使用余弦相似度（Cosine Similarity）来衡量两个嵌入向量之间的相似性。
• 根据欧氏距离是否小于某个阈值（例如1.1），判断两张人脸图像是否属于同一个人。

5.主程序逻辑
• 主程序通过调用`compare_faces`方法，比较两张人脸图像。
• 输出欧氏距离、余弦相似度和判断结果。

总体思路
• 加载模型：加载预训练的人脸识别模型，使其可用于后续的嵌入向量计算。
• 图像预处理：将输入的人脸图像调整为模型所需的格式。
• 嵌入向量计算：利用模型将人脸图像转换为嵌入向量。
• 相似度计算：通过计算嵌入向量之间的距离或相似度，判断两张人脸图像是否属于同一个人。
• 结果输出：输出比较结果，包括欧氏距离、余弦相似度和是否为同一人的判断。

关键点
• 预训练模型：使用预训练的深度学习模型是实现高效人脸识别的关键。这些模型通常在大规模人脸数据集上训练，能够提取人脸的特征。
• 嵌入向量：嵌入向量是模型的输出，它将人脸图像映射到一个高维空间，使得同一人的嵌入向量更接近，不同人的嵌入向量更远离。
• 相似度度量：通过计算嵌入向量之间的距离或相似度，可以量化两张人脸图像的相似性。
• 阈值判断：根据欧氏距离是否小于某个阈值，判断两张人脸图像是否属于同一个人。
这种思路在人脸识别领域非常常见，广泛应用于身份验证、人脸检索等场景。
